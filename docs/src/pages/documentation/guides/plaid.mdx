# Self-Hosted Plaid Access

Access your bank account via Plaid on a self-hosted monetr setup.

This guide will walk you through the steps required to connect your monetr instance to Plaid.

## Requirements

- A dedicated domain name or subdomain for monetr. For example: `monetr.myhomelab.com`
- TLS setup on that domain, Plaid requires HTTPS.
